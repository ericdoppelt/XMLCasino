View XML Page


The View XML file drives the construction of the specific game view. It has two meta tags which encode the game Title and file Authors. Additionally, it encodes the starting dimensions of the screen in the Width and Height tags.


The remaining tags encode information to compose the SettingsBar as well as visual effects internal to the Game. There are Language and Stylesheet tags which fill the ComboBox dropdowns on the top of the screen. Additionally, there is an IconBundle and ErrorStyleSheet which encode visual components of the game. All of these tags which point to specific data files/bundles are validated by enumerated types internal to the view schema.
